# Composition Calculator

**Composition Calculator** is a ratio-based calculator designed to help users easily compute and manage ratios for various applications, such as mixing solutions, recipes, or any scenario where proportions matter.

## Getting Started
Download the executable from the latest release and run!

## Usage

Input the desired ratios and a single quantity, and the calculator will provide the remaining quantities to use.

- The calculator outputs the required amount for each part.
